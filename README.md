# Premium Landing Experience

A complete frontend project built with vanilla HTML, CSS, and JS, structured with a modular folder architecture.

## Project Structure

```
/project-root
  ├── index.html        # Main entry point
  ├── README.md         # This file
  ├── src/
  │    ├── css/         # Modular styles
  │    │    ├── main.css
  │    │    ├── intro.css
  │    │    └── landing.css
  │    ├── js/          # Modular logic
  │    │    ├── main.js
  │    │    ├── intro.js
  │    │    └── landing.js
  │    └── assets/      # Static assets
  │         ├── videos/
  │         ├── images/
  │         └── fonts/
```

## Features

- **Intro Screen**: Fullscreen video background with START/SKIP transitions.
- **Landing Page**: 8 fullscreen (100vh) sections with smooth scroll-snap behavior.
- **Custom Navigation**: Fixed hamburger menu and interactive bottom number indicators.
- **Premium Design**: Colorful radial gradients, modern typography (Inter), and smooth micro-animations.

## How to Run

Simply open `index.html` in any modern web browser. No build steps or servers required.
